# #4451. [Cerc2015]Frightful Formula

内存限制：64 MiB时间限制：10 Sec

## 题目描述

给你一个n*n矩阵的第一行和第一列，其余的数通过如下公式推出： 

F[i,j]=a*f[i,j-1]+b*f[i-1,j]+c 

求f[n][n]%(10^6+3) 

## 输入格式

第一行三个数n，a，b，c 

第二行n个数，第i个表示f[i][1] 

第三行n个数，第i个表示f[1][i] 

2<=n<=200000 

其余的数大于等于0小于等于10^6 

## 输出格式

仅一个数表示f[n][n]%(10^6+3) 

## 样例

### 样例输入

    
    Sample Input1: 
    3 0 0 0 
    0 0 2 
    0 3 0 
    
    Sample Input2: 
    4 3 5 2 
    7 1 4 3 
    7 4 4 8 
    

### 样例输出

    
    Sample Output1: 
    0 
    
    Sample Output2: 
    41817
